我将把对话页空白状态下的星形图标替换为带有圆角矩形边框的 App 图标。

**修改计划**

1. **文件**: `app/src/main/java/com/fishai/chatzen/ui/screens/ChatScreen.kt`
2. **操作**:

   * 添加必要的 Import (`painterResource`, `border`)。

   * 定位到空白状态 UI 代码块 (约第 264 行)。

   * 将原有的 `Icon` (Icons.Default.AutoAwesome) 替换为 `Image` 组件。

   * 使用 `R.mipmap.ic_launcher` 作为图片资源。

   * 应用 `RoundedCornerShape` (圆角) 和 `border` (边框) 修饰符。

**代码预览**

```kotlin
Image(
    painter = painterResource(id = R.mipmap.ic_launcher),
    contentDescription = stringResource(R.string.ai_assistant),
    modifier = Modifier
        .padding(bottom = 16.dp)
        .size(64.dp)
        .clip(RoundedCornerShape(16.dp)) // 圆角
        .border( // 边框
            width = 1.dp,
            color = MaterialTheme.colorScheme.outlineVariant,
            shape = RoundedCornerShape(16.dp)
        )
)
```
