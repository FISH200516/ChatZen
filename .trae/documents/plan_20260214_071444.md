我将在模型提供商列表中添加 **海螺 (MiniMax)** 和 **英伟达 (NVIDIA)** 的支持，并配置为保存 API Key 后自动从官网获取模型列表。

**修改计划**

1. **添加文字资源** (`strings.xml`):

   * 新增 `provider_minimax`: "海螺 (MiniMax)"

   * 新增 `provider_nvidia`: "英伟达 (NVIDIA)"

2. **更新模型定义** (`ModelInfo.kt`):

   * 在 `Provider` 枚举中添加：

     * `MINIMAX`: API 地址 `https://api.minimax.chat/v1/`

     * `NVIDIA`: API 地址 `https://integrate.api.nvidia.com/v1/`

   * 更新 `ModelInfo.manufacturer` 属性，识别这两个新厂商。

3. **启用自动获取模型** (`SettingsViewModel.kt`):

   * 修改 `fetchAndSaveModels` 方法，将 `MINIMAX` 和 `NVIDIA` 加入到支持自动获取模型列表的提供商分支中。这样点击保存时，App 会自动向 `${baseUrl}models` 发起请求拉取最新模型。

**代码变更预览**

```kotlin
// SettingsViewModel.kt
val models = when (provider) {
    Provider.OPENAI -> { ... }
    // Add MINIMAX and NVIDIA here
    Provider.DEEPSEEK, Provider.SILICONFLOW, Provider.VOLCENGINE, Provider.MIMO, Provider.MINIMAX, Provider.NVIDIA -> {
        val url = "${provider.baseUrl}models"
        val response = RetrofitClient.openAIApi.getModelsUrl(url, "Bearer $trimmedKey")
        // ...
    }
    else -> emptyList()
}
```
