# 优化输出内容显示 (Markdown 支持)

为了实现聊天内容中 `**内容**` 等符号自动转化为加粗等富文本效果，我将引入 Markdown 渲染库。

## 1. 添加依赖
- 在 `gradle/libs.versions.toml` 中添加 `multiplatform-markdown-renderer-m3` (v0.39.2)。
- 在 `app/build.gradle.kts` 中应用该依赖。

## 2. 更新 UI 实现
- 修改 `app/src/main/java/com/fishai/chatzen/ui/screens/ChatScreen.kt`。
- 将 `ChatBubble` 组件中用于显示消息内容的 `Text` 组件替换为 `Markdown` 组件。
- 将 `ReasoningContentBlock` 组件中的 `Text` 组件也替换为 `Markdown` 组件（如果需要）。
- 使用 `CompositionLocalProvider(LocalContentColor provides textColor)` 确保 Markdown 内容颜色与当前气泡背景适配。

## 3. 预期效果
- 用户发送或接收的消息中的 Markdown 语法（如 `**粗体**`, `*斜体*`, `- 列表`, `Code block`）将被自动渲染为对应的样式。
